git 记住用户名密码:
永久记住: git config --global credential.helper store 
	会在用户主目录的.gitconfig文件中生成下面的配置。

	[credential]
		helper = store
	如果没有--global，则在当前项目下的.git/config文件中添加
	
临时记住密码
	默认记住15分钟：

	git config –global credential.helper cache
	下面是自定义配置记住1小时：

	git config credential.helper ‘cache –timeout=3600’
